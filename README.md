# CSTAR Media — Meta Ads Agency Automation System

**Niche:** Local service businesses (home services, health & wellness, automotive, beauty)
**Platform:** Meta (Facebook & Instagram) Ads
**Offer:** $1,500/month — 10+ qualified leads within 30 days or you don't pay
**Positioning:** "I run Facebook and Instagram ads for local service businesses to get them 10+ qualified leads per month within 30 days."

---

## Folder Structure

| Folder | What's Inside |
|--------|---------------|
| `/outreach` | Cold DM templates, prospect tracker |
| `/sales` | Call script, objection handling cheat sheet |
| `/delivery` | Campaign launch checklist, ad copy templates, weekly report |
| `/operations` | SOPs for every repeating task, client workspace template |

## Quick Start (New Client Checklist)
1. Run prospect research → `/outreach`
2. Send cold DMs → `/outreach/cold-dm-templates.md`
3. Book discovery call → `/sales/call-script.md`
4. Launch campaign → `/delivery/campaign-launch-checklist.md`
5. Send weekly reports → `/delivery/weekly-report-template.md`

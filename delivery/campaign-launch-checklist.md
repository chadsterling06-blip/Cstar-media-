## ACCOUNT SETUP
- [ ] Client has added you as partner in Meta Business Suite
- [ ] Ad account created or access confirmed
- [ ] Billing method added and verified
- [ ] Business Manager verified (no restrictions)

## PIXEL & TRACKING
- [ ] Meta Pixel installed on client website
- [ ] Pixel firing confirmed (use Meta Pixel Helper Chrome extension)
- [ ] Events set up: Lead, Contact, Schedule (whichever applies)
- [ ] Attribution window set: 7-day click, 1-day view
- [ ] Conversions API connected (if possible)

## NAMING CONVENTION
Use this format for everything:
- Campaign: `CSTAR_[ClientInitials]_[Goal]_[Date]`
- Ad Set: `CSTAR_[ClientInitials]_[Audience]_[Date]`
- Ad: `CSTAR_[ClientInitials]_[Creative#]_[Date]`

Example: `CSTAR_ABC_Leads_2026-04` / `CSTAR_ABC_Homeowners25-55_2026-04` / `CSTAR_ABC_V1Hook_2026-04`

## CAMPAIGN STRUCTURE
- [ ] One campaign per goal
- [ ] One ad set per audience segment
- [ ] 3–5 ad creatives per ad set
- [ ] Campaign budget optimization (CBO) ON for $50+/day budgets
- [ ] A/B test set up if budget allows

## AUDIENCE SETUP
- [ ] Primary audience defined (age, gender, location, interests)
- [ ] Lookalike audience created (if existing customer list available)
- [ ] Retargeting audience created (website visitors, video viewers)
- [ ] Exclusions added (existing customers if list available)

## CREATIVE CHECKLIST
- [ ] Hook (0–3s) grabs attention — pain point or bold statement
- [ ] Hold (3–15s) keeps interest — problem agitation or credibility
- [ ] Offer clearly stated
- [ ] CTA clear and specific ("Call now," "Get a free quote," "Book today")
- [ ] All text reviewed for grammar/spelling
- [ ] All images/videos reviewed — no blurry, dark, or unprofessional assets
- [ ] Mobile preview checked in Ads Manager

## BASELINE METRICS (record on day 1)
- CPM (cost per 1,000 impressions): $
- CTR (click-through rate): %
- CPC (cost per click): $
- Target CPL (cost per lead): $

## FINAL CHECKS
- [ ] All ads submitted and approved by Meta
- [ ] Client briefed on what to expect in first 48–72 hours
- [ ] Client reminded: DO NOT JUDGE results in first 48–72 hours (algorithm learning)
- [ ] First weekly report date scheduled (Monday after launch week)
- [ ] Client added to weekly report tracker

---

## IMPORTANT: 48–72 HOUR RULE
Do NOT touch the campaign in the first 48–72 hours after launch.
The algorithm is in the learning phase. Editing resets it.
Wait. Then optimize.
